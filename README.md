# Mad Shade (IVYUSA Sub brand) #

From Amoeba DevOps Team

## Requirements dev

```
- Installed webserver (Apache, nginx)
- Installed composer
- Installed node
```

## Environment

```
- PHP >= 7.4
- Wordpress >= 6.2
- Mysql MariaDB 10.8
---
- Nodejs 16.13.0
- Npm 8.1.0
```

## Run
```
- composer install
- gulp watch
```
